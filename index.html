Boeing: Lead a team of seven engineers in the Propulsion Integration Lab where Engine Controls Software is tested against thousands of complex integration and software requirements. I am also spearheading the automation effort in the lab with the end goal of reducing test time, increase testing consistency, and providing complete coverage of requirements.

In addition to leading lab testing, I manage Engine Control's related Flight Test Problem Reports and the Interface Control Document that contains engine software requirements agreed to between Boeing and suppliers.

Some projects I am working on or completed:

  • Developed automated testing framework for engine software testing that can be applied to all future airplane programs.
  • Developed software to analyze requirements to test plan linking, identifying missing verifications and changes across thousands of requirements, allowing for 100% requirement coverage.
  • Created VBA scripts to process and analyze Problem Reports pulled from the flight test databases, automating a previously manual process.
  • Created tools in Matlab for fuels flow analysis around the aircraft, utilizing GitLab for configuration management. Developed a GUI to import and adjust the fuel piping system for Simulink analysis. Greatly increased the accuracy of the automatically generated Simulink models.
  • Developed an augmented reality Android app to overlay a virtual image on the 777X wing spar to assist manufacturing and maintenance activities.
